Morse code study

Learn to recognise Morse code signals and identify letters as fast as you can. No prior experience needed — that is the whole point!

Condition locked to Audio + Visual for sessions 3 and 4.
📖

How it works

Morse code uses two types of signals — a short one and a long one. Each letter is a unique combination of these two signals.

Short signal — the Dot
Long signal — the Dash

The 10 letters you will learn

Each chip shows the letter and its Morse pattern:

E  . T  - I  .. A  .- N  -. M  -- S  ... R  .-. K  -.- F  ..-.

What to do each trial

A signal will play automatically. When it finishes, type the letter you think it was and press Enter. Respond as fast as you can — guessing is absolutely fine!

After each answer you will see whether you were right and what the correct pattern was, so you can learn as you go.

🎮

Practice round

5 warm-up trials to get used to the format. These are not recorded — no pressure!

Remember: type the letter and press Enter after each signal. The Morse pattern will be hidden when you answer so you have to rely on what you heard or saw.
